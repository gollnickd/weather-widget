== Description ==

The Perfect Paddles Water Conditions Widget displays real-time water and wind conditions with color-coded difficulty levels (Beginner, Intermediate, Advanced) for watersports activities like paddleboarding, kayaking, and canoeing.

Features:

* Real-time weather and wind data
* Color-coded condition indicators (Green/Orange/Red)
* Automatic 10-minute refresh intervals
* Beautiful, responsive design
* Perfect Paddles branding
* Easy installation via shortcode or widget

Perfect for:
* SUP rental businesses
* Kayak tour companies
* Marina websites
* Beach activity providers
* Watersports schools

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/pp-water-widget/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings â†’ Water Conditions Widget
4. Enter your API key (obtain from Perfect Paddles)
5. Use the shortcode `[pp_water_widget]` or add the widget to a sidebar

== Frequently Asked Questions ==

= How do I get an API key? =

Contact Perfect Paddles at support@perfectpaddles.com to obtain your API key.

= How often does the data update? =

The widget automatically refreshes every 10 minutes with the latest conditions.

= Can I customize the appearance? =

The widget uses Perfect Paddles branding. For custom styling, please contact us.

= What determines the condition level? =

* Beginner (Green): Wind < 10 mph, calm water
* Intermediate (Orange): Wind 10-18 mph, moderate chop
* Advanced (Red): Wind > 18 mph, challenging conditions
